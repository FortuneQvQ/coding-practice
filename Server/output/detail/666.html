我院与韩国延世大学牙学院续签合作备忘录
发布时间：
重点新闻 院内动态 信息公开 2026年7月7日，韩国延世大学牙学院院长Jung Young Soo教授率代表团一行5人到我院访问交流，并与我院续签合作备忘录。我院院长韩向龙教授、副书记林云锋教授、副院长袁泉教授及相关职能部门负责人参加座谈交流。 座谈会上，韩向龙院长代表我院对代表团表示热烈欢迎。他表示，双方合作基础深厚，自2018年签署首轮合作备忘录以来，双方在师生互访、学术交流、科学研究等方面开展了富有成效的合作。Jung Young Soo院长对此高度认同并积极回应，提出期待与我院持续拓展、深化长期合作伙伴关系。座谈结束后，双方代表续签新一轮合作备忘录。双方将以此次续签为契机，进一步深化在人才培养、科学研究及师生交流等方面的合作，共同推进口腔医学事业的发展。  访问期间，代表团还参观了华西口腔医学博物馆、临床科室及口腔疾病防治全国重点实验室。 上一篇：我院入选2026医学创新转化领军型医院 下一篇：我院召开树立和践行正确政绩观学习教育警示教育会 // jQuery(function () { if ($('#left_1')) { $("#left_p_1").addClass("current");
