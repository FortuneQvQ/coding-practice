我院正颌及关节外科党支部与宁夏医科大学总医院口腔医院口腔颌面外科党支部联合开展主题党日活动

思想建设 组织建设 作风建设 党风廉政 综合信息 党纪法规 廉洁教育 专项工作 工作流程 办事指南 统战工作 群团工作 群团工作 团员青年 职工之家 新闻动态 理论学习 通知公告 2026年6月27日，我院正颌及关节外科党支部赴宁夏医科大学总医院口腔医院，与其口腔颌面外科党支部联合开展“深学实干精神，树牢正确政绩观”主题党日活动，牢固树立为民务实的正确政绩观。 支部党员首先前往宁东科创基地，了解基地产业升级、科技创新发展历程，学习建设者攻坚克难、实干担当的优良作风。随后，支部党员聚焦临床诊疗能力提升、就医流程优化、患者术后康复管理等相关问题，展开深入交流讨论。大家一致表示，将在临床工作中发扬实干精神，摒弃形式主义，稳步推进诊疗服务、学科建设和公益医疗事业。 下一步，双方将持续深化联建共建机制，常态化开展理论学习和经验交流，以过硬作风和务实举措持续践行正确政绩观，推动口腔医学学科高质量发展。 上一篇：我院口腔疾病防治全国重点实验室第一党支部开展树立和践行正确政绩观学习教育专题党课 下一篇：我院扎实开展树立和践行正确政绩观学习教育专题党课 // jQuery(function () { if ($('#left_1')) { $("#left_p_1").addClass("current");
