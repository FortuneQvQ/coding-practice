项目发布 |【大川视界】德国亚琛工业大学2025年 暑假学分项目
发布时间：2025-03-13
为提升学生的国际视野，推进我校人才培养国际化战略，促进国际传播力建设和提升学校的国际影响力，学校现启动选派优秀本科生和研究生赴德国参加大川视界“亚琛工业大学2025年暑假学分项目”的工作。为保证此项工作的顺利进行，现就有关事项通知如下： 一、项目简介 （一）学校简介 德国亚琛工业大学（RWTH Aachen University）成立于1870年，是世界顶尖理工科大学之一，被誉为“欧洲的麻省理工”。现为11所德国“精英大学”之一，4所欧洲顶尖理工类大学的战略联盟（IDEA联盟）、9所德国理工大学联盟（TU9联盟）之一、TIME欧洲顶尖工业管理者高校联盟德国七所高校之一。该校理工科实力强劲，电子、机械制造、冶金工程方面排名德国第一，计算机、建筑、物理等学科排名德国前三， 2025 QS世界大学排名第99位，2024泰晤士高等教育世界大学排名第90位。 （二）暑假课程 亚琛工业大学2025年暑假学分项目提供四个主题班课程，每个班25-30名学生。本科生和研究生均可报名。 主题班1： 先进制造领域的数字化(DIM) Digitalization in Manufacturing 主题班2： 数据分析与Python编程（DAPP）Data Analytics and Python Programming 主题班3： 工业4.0智能制造工程（SPEI）Smart Production Engineering in Industry 4.0 主题班4： 车辆工程创新技术（AM）Innovative Technologies in Automotive Engineering 课程介绍请见附件。 （三）项目优势和特色 在顶尖理工大学学习工科领域前沿知识 体验跨⽂化交流 考核合格可获得3个欧洲学分ECTS 将由亚琛工业大学签发官方结业证书 二、项目时间和费用 主题班 时间 费用 主题班1: 先进制造领域的数字化(DIM) Digitalization in Manufacturing 7 月12日-7月25日 3190 欧元 ( 约合人民币2.4万） 主题班2: 数据分析与Python编程（DAPP）Data Analytics and Python Programming 7 月28日-8月10日 3190 欧元 ( 约合人民币2.4万） 主题班3： 工业4.0智能制造工程（SPEI）Smart Production Engineering in Industry 4.0 8 月11日-8月24日 3190 欧元 ( 约合人民币2.4万） 主题班4： 车辆工程创新技术（AM）Innovative Technologies in Automotive Engineering 8 月11日-8月24日 3990 欧元 含每人一个课上搭建并可带走的乐高小车 ( 约合人民币2.99万） 费用包含：教学、参观相关资源及活动安排，亚琛住宿（ 2 人间），周一至周五早餐和午餐，亚琛当地公交车票，相关行政服务与支持等。 费用不包含：机票、保险费、签证费、不包含的膳食以及个人消费。 三、项目报名条件 （一）拥护中国共产党的领导、热爱祖国、品德优良；遵章守纪，无纪律处分； （二）全日制非毕业年级在校本科生和研究生，专业不限（出发前须年满18周岁）；大一学生不得影响军训。 （三）语言要求：适应全英文授课（无硬性分数要求，可提供各类英语考试成绩供参考）。 （四）综合素质：具备开放包容的交流心态和独立海外生活能力；身心健康；学生在国外学习期间，尊重当地风俗，遵守当地法律法规和双方学校的规章制度，接受双方学校的管理。 四、项目申请时间与申请方式：两校同时报名 （一）校内申请时间： 即日起至3月28日 （二）校内申请流程： 1. 本科生 登录教务处国内国际合作交流管理系统http://jxyw.scu.edu.cn:8081/gjjl/，选择“德国亚琛工业大学2025年暑假学分项目（“大川视界”）”进行在线申请。 特别提醒： （1）请下载登录界面操作手册后进行操作，若遇技术问题，可打电话13183867686或QQ：1134372115咨询。 （2）为实现项目报名工作流程化、信息化，学院、学校将通过系统完成项目审批（报名学生无需再交纸质材料），请学生在线报名后及时告知学院教务办，请学院相关人员在系统中于截止日期内完成学院审核。 （3）川大所修课程中文成绩以系统计算为准，若有疑问，请持成绩单原件与教务处合作与交流科联系。 2. 研究生 通过研究生院申请 （1）申请人：登录四川大学研究生管理系统——国际学术交流申请应用，选择“寒暑假访学项目申请”，选择相应的访学项目，填写申请，提交并打印出纸质《申请表》，将纸质《申请表》递交导师、学院审核。 （2）导师：登录四川大学研究生管理系统——国际学术交流应用，在“寒暑假访学项目”中审核并在纸质版《申请表》上签署意见、签字。 （3）学院：学院研究生教务老师登录四川大学研究生管理系统——国际学术交流应用，在“寒暑假访学项目”中审核，并由学院主管领导在纸质版《申请表》上签署意见、签字、盖公章。 （4）学院审核后，纸质《申请表》以学院为单位报研究生院3-325培养办公室。 （5）申请人回国后，登录四川大学研究生管理系统——国际学术交流应用，在“寒暑假访学项目”申请页面中填写并提交回国总结及以下材料： ① 回校证明（包含项目名称、访学开始和结束时间并由辅导员老师签字） ② 官方颁发的项目证书或成绩单 （三）因主题班课程名额有限，为帮助同学们尽快申请到心仪的课程，满足报名条件的同学在校内报名的同时，可同步向亚琛工大申请，由亚琛工大根据报名先后和语言成绩综合录取。只在外方院校报名而未在校内报名的同学，将不能申请“大川视界”资助。
