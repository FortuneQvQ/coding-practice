我院入选2026医学创新转化领军型医院
发布时间：
重点新闻 院内动态 信息公开 2026年7月3日至5日，第八届CMIA医学创新大会在青岛召开，我院凭借在医学创新转化工作中的卓越表现，入选2026医学创新转化领军型医院，位居全国医疗机构创新转化第一方阵。 本届大会特设2026医学创新瑯琊汇典型案例征集活动，我院2项成果从全国脱颖而出，医院报送的“从院企专项基金到院企地生态共创：口腔医学四链融合创新生态的迭代升级之路”获评“体系生态创新类卓越实践”案例，韩向龙教授团队报送的“错（牙合）畸形精准诊疗体系的关键技术创新团队”获评“卓越人物”案例。 入选CMIA全国医疗机构创新转化第一方阵，是对我院长期以来在产学研体制机制建立、创新平台构建、加强知识产权保护与转化运用等方面积极探索，以全链条知识产权服务助力高质量发展的肯定。今后，我院将继续推进产学研合作，为推动我国口腔医药、设备、材料及器械产业创新发展贡献力量。 上一篇：新华网、《人民日报》、中国国际电视台等多家媒体报道我院医疗队赴莫桑比克开展“微笑… 下一篇：我院与韩国延世大学牙学院续签合作备忘录 // jQuery(function () { if ($('#left_1')) { $("#left_p_1").addClass("current");
