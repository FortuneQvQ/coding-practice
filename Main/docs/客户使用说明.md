# 校园资讯聚合系统：客户使用说明

## 一、普通访客如何使用

1. 打开管理员提供的“校园资讯聚合系统”网址。
2. 首页展示最新资讯，点击“查看详情”阅读完整正文和已归档图片。
3. 在首页搜索框输入一个或多个关键词，点击“搜索”。系统会检索标题、正文、摘要、来源和主题。
4. 点击“查看更多新闻与分类”，可以按时间、来源或主题浏览：
   - 来源：教务处、学生会、学院通知、学生社团。
   - 主题：教学通知、校园活动、竞赛比赛、考试安排、科研创新。
5. 在详情页可返回搜索结果或分类结果，也可以点击“官网原文”核对信息。

## 二、本机演示如何启动

本机需要已安装 Python 3（启动脚本会自动查找 `python.exe` 或 `py.exe`）。

1. 双击项目目录中的 `启动网站.bat`。
2. 浏览器会自动打开 `http://127.0.0.1:8765/`。
3. 保持黑色服务窗口开启；关闭该窗口即可停止网站。

不要直接双击 `output/index.html`：搜索和分类功能需要通过本地网站地址读取 `json/news.json`。

## 三、管理员如何更新内容

- 正常更新：双击项目目录中的 `运行更新.bat`。窗口会保持显示，程序依次爬取新资讯、去重入库、归类、下载新图片并重新生成网站。
- 唯一新闻数据库位于项目根目录的 `data\news.db`；同目录的 `news.db-wal` 和 `news.db-shm` 是 SQLite 运行时辅助文件，不是重复数据库。
- 当前 Release 程序位于项目根目录的 `bin\x64\Release\CampusNewsSystem.exe`。
- 只重新生成展示网站：运行 `bin\x64\Release\CampusNewsSystem.exe --generate-only`，不会爬取网页或消耗种子配额。
- 只验证爬取能力：运行 `bin\x64\Release\CampusNewsSystem.exe --scan-only`，不修改新闻表和游标。
- 暂不生成网站：运行 `bin\x64\Release\CampusNewsSystem.exe --no-website`。

正常更新后，网站文件位于 `output`：

- `index.html`：首页。
- `detail`：新闻详情页。
- `images`：从原网站归档的正文图片。
- `json/news.json`：搜索和分类使用的数据。
- `js`：展示模块脚本。
