第17届全国现代逻辑学术研讨会征稿启事（第二轮）
发布时间：2023-05-25
第 17 届全国现代逻辑学术研讨会 征稿启事 （第二轮） 请见第三轮征稿启事： https://zxx.scu.edu.cn/info/1016/3060.htm 第 17 届“全国现代逻辑学术研讨会”计划于 2023 年 10 月 20-22 日在四川省成都市召开。本届研讨会由中国逻辑学会现代逻辑专业委员会主办，四川大学哲学系和四川大学逻辑、科学与文化研究所承办，四川省逻辑学会协办。会议包括 ( 但不限于 ) 特邀报告、一般学术报告、学术海报展示等环节；其中特邀报告人为 ( 拼音字母序 ) ：丁龙云 ( 南开大学数学科学学院 ) 、刘咏梅 ( 中山大学计算机学院 ) 、张德学 ( 四川大学数学学院 ) 。会议不收 取 注册费；参会者需自理差旅和食宿。 会议投稿通道已经开放 https://easychair.org/conferences/?conf=ncml23 。欢迎满足如下条件的学术论 文或长摘要稿件：  主题与现代逻辑有密切联系 ( 包括但不限于以下方向：哲学逻辑，数理逻辑，中外逻辑史，逻辑哲学，逻辑学与语言学、计算机科学或其他学科的交叉 ) ；  符合学术规范，有具体原创结果，尚未发表，没有同时投稿到国内外正式刊物或学术会议；  使 用 中文或英文其中一种写作， 篇幅 不超过 15 页 [①] ;  由通讯作者通过 easychair 平台投稿通道 准确填写所有作者的信息，并上传稿件的 PDF 格式电子文档 ( 可由 LaTeX 或 MS Word 等工具生成，可使用推荐的模板 [②] ) ； 所有作者的姓名、单位、资助及其他可明显揭示作者身份的信息已在电子文档中被隐去。 程序委员会将对每篇通过初筛的稿件以匿名方式进行评审 ( 审稿人对作者匿名、作者对审稿人匿名、审稿人之间互相匿名 ) ，并依据稿件质量做出如下审稿结论之一： （ A ）接收为一般学术报告、（ B ）接收为学术海报展示、（ C ）不接收。 程序委员会成员包括 ( 拼音字母序 ) ：丁一峰、董惠敏、范杰、方良达、高坤、郭佳宏、郭美云、郝兆宽、何家亮、贾青、琚凤魁、李延军、梁飞、林翰迪、林哲、刘婵娟、刘虎、刘佶鑫、刘明明、刘新文、任远、荣立武、申国桢、沈榆平、石辰威、苏庆辉、孙鑫、王文方、王彦晶、王轶、文学锋、邢滔滔、熊明、熊作军、徐召清、杨帆、杨睿之、杨跃、姚宁远、叶峰、俞珺华、喻良、张寄冀、张立英、张炎、赵希顺、赵贤、钟盛阳。 每篇获得（ A ）或（ B ）结论的稿件，将由其至少一名作者亲自 注册参会，并参与相应的公开报告或公开展示环节；这些稿件也将 被收录于会议的印刷版论 文 集中 ( 不是正式出版物，且不会以其他形式公开 ) 。获得（ A ）结论的稿件中，质量优秀且在会公开报告环节正常的，将在通讯作者征得所有作者同意的前提下，被推荐给 CSSCI 源刊《逻辑学研究》杂志 ( 本会议审稿结果不替代该刊对稿件做出的决定 )( 特别感谢中 山 大学《逻辑学研究》编辑部的 大力支 持 ！ ) 。 重要时间 ( 均为 2023 年 ) ： 投稿截止 07月23日 （已延迟到7月31日） 审稿结论通知 09 月 10 日 被接收稿件修改版定稿 0 9 月 25 日 有任何问题，请联系： 杨睿之 ( yangruizhi@fudan.edu.cn ) 、俞珺华 ( junhua.yu.thu@139.com ) 。 [ ① ] 包括参考文献在内。稿件可后附一节不计入篇幅限制的“附录”用于展示正文中略去的细节，但这不应影响稿件正文部分的完整性；该 “ 附录 ” 在审稿环节中由每位审稿人独立自愿选读，不构成对稿件质量的支持，且不会被收录在会议文集中。 [ ② ] 感谢《逻辑学研究》编辑部和王轶老师制作模板！下载链接 - https://share.weiyun.com/XZYSogHU 密码 - a8ht6x
