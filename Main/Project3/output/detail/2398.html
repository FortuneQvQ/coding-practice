关于收集2021-2022学年秋季学期本科课程考核材料的通知
发布时间：2021-12-08
本学期开设本科生课程的各位老师： 临近期末，请您按学校教务处关于课程成绩登录的通知（https://jwc.scu.edu.cn/info/1081/8198.htm）按时录入成绩之外，将下列材料（1-9）准备好， 最迟于202 2 年春季学期第 二 周周五之前（2022年3月 4 号前）交回江安校区学院522教务办公室装盒存档 （望江文科楼学院教务办公室不再接收存档材料）： 1、课程考核方案一份（请 下载附件1 ）； 2、空白试题一份（有AB卷的请交齐AB卷）、 评分标准（适用于课程论文）或参考答案（适用于有客观题的）一份； 3、考场签到表、考场记录表（期末教室卷面考试的提供） 4、 学生答卷或纸质课程论文； 5、 课程成绩登记表（成绩录入系统中导出打印并在下方签字）； 6、 课程成绩分析表（成绩录入系统中导出打印填写并签字）； 7、 试卷评分复查表（除开任课教师之外的系教研室其他教师填写，请 下载附件2 ）； 8、 过程性考核材料（如平时作业、考勤等平时成绩原始记录材料）； 9、学生缓考申请表（若有缓考申请的需提供） 未尽事宜，请致电028-85996613（赵）咨询。 温馨提示： 附：评卷要求： 1、试卷红笔批改，有客观题的，各题扣分处明确标示出“负分”，在各小题处标记本题得分，即“正分”，卷头记卷面总分，并须任课教师签名； 2、课程论文、报告式作业批改需用红笔在封面记总分并写200字以内的评语，任课教师或指导教师签名并落款日期； 3、凡有改动分数的地方必须有任课教师签名。 附件1： file 附件2： file
