四川大学历史文化学院研究生教务管理岗招聘
发布时间：2023-09-28
一、岗位待遇 : 参照学校相关规定执行 二、经费来源 : 学院经费   三、招聘人数： 1人 四、招聘截止日期： 202 3 年10 月30日 五、岗位职责 : 1.   对口校研究生院招生办，负责学院招生前期准备、复试、录取等硕博研究生招生工作。 2.   承担学籍管理、教务管理、毕业资格审查等学院研究生培养工作。 3.   协助教务办科长完成研究生办公室日常工作。 4. 担任学院外事秘书，承担学院师生出入境审核、日常涉外管理、外事接待、项目申请、更新新闻网站等学院外事工作。 5 .   完成学院领导交办的其他工作。   六、招聘条件 : 1.   政治立场坚定，具有较高的政治素质和理想信念； 2.   新进校人员应具有国内外高水平大学硕士学位，身心健康，年龄不超过 30岁，并在2022年12月31日前取得硕士学位； 3.   工作踏实认真，爱岗敬业，具有强烈的事业心、责任心和服务意识； 4.   具备较强的组织能力，协调能力，沟通能力与合作意识； 5.   具备较强的写作能力和语言表达能力，能熟练运用相关办公软件； 6.   为人正直，作风正派，无违法违纪行为记录。 七、招聘程序 : 1.   应聘人员将个人详细简历发送至邮箱 renshi85412314@163.com。 2.   学院组织人事科进行资格初审； 3.   学院新进人员选聘工作小组对初审通过的应聘人员进行甄选，确定参加面试人员名单； 4.   单位新进人员选聘工作小组对候选人进行面试，择优确定拟聘人员； 5.   拟聘人员进行体检、公示，无异议后上报学校； 6.   将拟聘人员报学校审批，审批通过后签订合同、办理报到手续。
