四川大学【2026-2027】年度水池水箱清洗消毒服务遴选公告
发布时间：2026-07-14
为保障校园二次供水饮用水安全，四川大学后勤保障部维修与通讯服务中心现就【2026-2027】年度校区水池水箱清洗消毒服务项目进行公开遴选采购，本项目采用满足采购需求前提下最低价中标方式确定服务商，欢迎符合资格条件的供应商踊跃报名参与。 一、项目概况 1. 服务期限：【2026年8月—2027年8月】，服务周期 1 个自然年度，项目内所有水箱全年清洗消毒 2 次。 2. 服务地点：【望华校区、江安校区】（每年按需增减校区）。 3. 项目控制预算：大写：拾肆万玖仟元整，小写￥14.9万（每年更新预算金额）。 4. 服务内容：对项目清单内二次供水水池、屋顶水箱开展年度两次清洗消毒；按卫健及疾控部门要求分季度开展水质检测并出具检测报告；中标单位全权代办二次供水卫生许可证及相关报备手续；具体水箱点位、数量、容积详见招标文件附件点位清单（每年更新点位数据）。 二、供应商资格条件 1. 具有独立法人资格，能够独立承担民事责任； 2. 具备良好商业信誉及健全的财务会计管理制度； 3. 具备水箱清洗、消杀作业必备设备、专业技术人员及履约能力； 4. 依法按时足额缴纳税收和社会保障资金，无不良缴纳记录； 5. 参加政府采购活动近三年内无重大违法经营记录（须提供书面声明）； 6. 符合国家法律法规规定的其他相关条件。 三、报名所需资料（所有复印件加盖供应商鲜章） 1. 营业执照、税务相关证件复印件； 2. 水箱清洗、消毒消杀相关资质证书复印件； 3. 单位介绍信、项目经办人有效身份证复印件。 四、报名及开标相关事项 1. 报名截止时间：【2026年7月17日17:00（星期五）】，逾期不再接收报名资料； 2. 资料提交方式：报名资料扫描件发送至邮箱：191794454@qq.com 3. 开标时间：【2026年7月20日下午15点（星期一）】；（如有调整，另行通知） 4. 开标地点： 后勤保障部2楼220评标室 5. 收到报名成功的商家需在开标当日14：45分携带报名资料纸质版（不密封并按“报名资料”要求准备），报价单纸质版（密封并加盖鲜章）及法人及授权委托人（如需）有效身份证正反面复印件及授权委托书（如需）（不密封并加盖鲜章）送至成都市武侯区一环路南一段24号四川大学望江校区后勤保障部二楼220评标室。 6. 本项目仅接受现场提交资料，不接收邮递形式。 五、联系方式 联系人：廖老师 联系电话：85404112 采购单位：四川大学后勤保障部维修与通讯服务中心 发布日期：【2026年7月14日】
